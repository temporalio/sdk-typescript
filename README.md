# Temporal NodeJS SDK

## This is a work in progress, not ready for use yet

For more information see the [proposal](https://github.com/temporalio/proposals/blob/master/node/node-sdk.md).

### Development

#### Environment set up

```
git submodule init
git submodule update
npm install
```

#### Building

```
npm run clean
npm run build
```

-- OR --

```
npm run clean
npm run build.watch
```

#### Testing

```
npm run test
```

-- OR --

```
npm run test.watch
```
